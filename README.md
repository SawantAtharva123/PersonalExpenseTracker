A simple Personal Expense Tracker Web Application built using the MERN stack that helps users manage and visualize their daily expenses efficiently. The application allows users to record transactions and analyze their spending through graphical insights.

This project was developed to practice full-stack development concepts, including frontend UI design, backend APIs, and database management.

🚀 Features

➕ Add new expenses with details like amount and description

📊 Visualize expenses using charts

🗑️ Delete or manage existing expenses

💾 Store expense data in a database

📱 Simple and responsive user interface

📈 Track spending patterns over time

Expense tracking applications help users monitor spending and understand financial habits through recorded transactions and visual summaries.

⚙️ Installation and Setup
1️⃣ Clone the repository
git clone https://github.com/SawantAtharva123/PersonalExpenseTracker.git
2️⃣ Navigate into the project
cd PersonalExpenseTracker
3️⃣ Install dependencies

Backend

cd server
npm install

Frontend

cd client
npm install
4️⃣ Run the application

Backend

npm start

Frontend

npm start

The application will run on:

http://localhost:3000
